# CNews
APP包含各種論壇基本功能，私信、加好友、收藏等等功能，利用Cordova套件，同時產出android版以及IOS版APP，達成雙平台的功用。
在此專案我擔任組長分配其他6名組員工作及使用"timetree"去做共用行事曆監督每個人作業，並且同時擔任APP UI設計。
